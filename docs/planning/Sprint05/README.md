# Sprint 5 - Device Enrollment

---

# Sprint Overview

Sprint 5 begins the practical implementation of the Microsoft Intune Enterprise Deployment project.

The objective of this sprint is to configure and validate Windows device enrollment within Microsoft Intune.

Successful completion of this sprint will confirm that Windows devices can be enrolled into Microsoft Intune, managed successfully, and prepared for future policy, application, and security deployments.

---

# Sprint Goal

Successfully enroll and validate the first Microsoft Intune managed Windows device.

---

# Sprint Objectives

- Review enrollment settings.
- Configure enrollment prerequisites.
- Enroll the first pilot device.
- Validate successful enrollment.
- Document implementation results.
